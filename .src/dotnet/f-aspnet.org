#+TITLE: ASP.NET



* ASP.NET

- JSP: Java Server Pages
- ASP: Active Server Pages
- ASP.NET

ASP.NET 为 .NET 平台提供了完整的进行 Web 开发的底层框架:
- (ADO.NET - EF)
- Webforms - Winforms (事件驱动、基于控件)
- ASP.NET MVC (模型-视图-控制器)
- ASP.NET Web API

ASP.NET core:
- Razor Page
- Blazor (组件，使用 C# 写前端)
- MVC (跨平台，内置 web 服务器，内置 DI 容器)
- Web API
